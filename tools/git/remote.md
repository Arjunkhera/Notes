## Remote repositories

### Cloning and stuff

```bash

// cloning a repository
git clone url optionaltargetdirectoryname

```

---

### Common commands

```bash

// list remote repositories
git remote

// adding a remote repository to track
git remote add givenametobranch urlofrepository

// pushing into a remote repository
git push remotename nameofbranchtobepushed

// using u as flag sets the given remote as default
git push -u remotename nameofbranchtobepushed
// so next time all you have to do is
git push

```
